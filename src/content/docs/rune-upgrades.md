本頁整理 2026-05-20 觀測到的 rune upgrade 資訊，方便查 `Val` 到 `Fyrm` 的 required level、升級材料與鑲嵌效果。完整策略與配置取捨請搭配 [符文（Runes）指南](./runes/) 閱讀；本頁只做資料查表，不把高階 rune 視為一定要從低階 rune 一路合上去。

- 檢視日期：`2026-05-20`
- 資料來源：遊戲 live 頁面 `Craft` tab DOM snapshot、`Val Rune` 截圖確認、`ngo.bin` readable string check
- 覆蓋範圍：`Val` 到 `Fyrm` 共 33 顆 rune；其中 `Val` 是基礎 rune，不在 crafting DOM 的可製作清單中
- 版本提醒：rune upgrade recipe、tooltip 文字、cost 與可 craft 狀態可能隨更新調整；投入高階 rune 前請以目前遊戲內 UI 為準

> **快速重點**
> `Shir` 到 `Thex` 需要 3 顆前一階 rune。
> `Marr` 到 `Fyrm` 需要 2 顆前一階 rune。
> 雙手武器效果通常是單手 / 項鍊效果的 2 倍；盾牌與防具同用防具效果判讀。

## 使用筆記

`Val` 不是從 Craft tab 製作出來的 rune，而是本次資料中以截圖補確認的基礎階。`Shir` 到 `Fyrm` 則來自 crafting DOM 中可見的 32 筆 rune upgrade。

高階 rune 的 immediate recipe 看起來比前半段便宜，但實際成本仍很高，也可能直接從掉落、交易或其他來源取得。查表時請把「上一階需要幾顆」理解為目前 Craft tab 顯示的單步 recipe，不要把它解讀成唯一取得路線。

效果文字已依站內繁中術語做輕度整理：`血` 類屬性寫成 `出血`，`抵抗` 寫成 `抗性`，擊殺 / 受擊回復則改成玩家較容易讀的語序。數值、rune 順序與 recipe count 仍依來源資料保留。
