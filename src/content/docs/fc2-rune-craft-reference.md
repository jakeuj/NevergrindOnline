<a id="fc2-itemmods-hdr1"></a>

### Damage 強化

這是一個僅適用於武器的MOD，並且加強了武器的傷害。裝備上顯示的數字和DPS是應用傷害增強後的數字和DPS。

<a id="fc2-itemmods-hdr2"></a>

### 攻擊速度增加

這是一個只能應用在武器上的MOD，可以提高武器的速度。裝備上顯示的數字和DPS是加上攻擊速度提升後的。

<a id="fc2-itemmods-hdr3"></a>

### 格擋率

這是一個僅適用於盾牌方塊的模組，並指示方塊的擴展率。這個數字只是命中率，格擋時的基礎傷害削減率為25%。另外，對於坦克職業，折舊率將增加20%。

<a id="fc2-itemmods-hdr4"></a>

### 格擋率增加

這是一個僅適用於盾牌瓷磚的模組，並加強了掉落率。設備上顯示的折舊率是折舊率增加後的。

<a id="fc2-itemmods-hdr5"></a>

### 物理防禦強化

這是只適用於護甲、強化物理防禦的MOD。裝備上顯示的物理防禦是應用物理防禦強化後的。

<a id="fc2-itemmods-hdr6"></a>

### 天賦/天賦

增加角色對應的天賦。如果有多個天賦，則不會抽取相同的天賦。它通常附著在武器、盾牌、副斗篷、頭部、軀幹、斗篷、腰帶、項圈和戒律上。 天賦是天賦的機器翻譯，gift是天賦的機器翻譯，唯一的差異是是否為複數。

- 擁有天賦
  培養各行各業的天賦。
- 完整性 職業名稱 天賦
  增加相關專業的各類天賦。
- 天然樹
  增加適用天賦樹中的所有天賦。
- 屬性天賦
  流血、毒素、奧術、閃電、火、冰。增加各職業對應屬性的天份。對於具有多種屬性的技能，只會應用一種。例如德魯伊的龍捲風是冰寒和奧術技能，但是只應用了冰寒天賦。
- 天賦
  增加相應的天賦。

<a id="fc2-itemmods-hdr7"></a>

### 狀態

全能力值、力量、耐力、敏捷、靈巧、智力、智慧、魅力。

|力量|物理技能傷害與 Attack Rating 提升；對物理技能傷害影響很大。|
|---|---|
|耐力|最大 Health 增加。|
|敏捷|閃避率與物理防禦提升。|
|靈巧|Attack Rating、招架、反擊、格擋率與暴擊率提升；Attack Rating 的提升幅度不如力量。|
|智力|最大 Spirit 與 Conjuration 法術強度提升；也會小幅影響 Evocation 法術強度。|
|智慧|最大 Mana 與 Evocation 法術強度提升；也會小幅影響 Conjuration 法術強度。|
|魅力|最大 Spirit 與 Alteration 法術強度提升。|

<a id="fc2-itemmods-hdr8"></a>

### Passive

全 Passive Skill、攻擊、防禦、單手斬擊、單手鈍器、刺擊、弓術、空手、雙手斬擊、雙手鈍器、雙持、Double Attack、閃避、招架、反擊、Alteration、Conjuration、Evocation。

武器被動非常重要，它極大地影響物理技能傷害和命中率。粗略來說，如果武器被動為15，物理技能傷害將增加3.3%。

法術被動不會增加那麼多法術傷害，所以有點奇怪。改變傾向於影響支持類型，例如恢復，咒語傾向於影響範圍類型，召喚傾向於影響物理類型。

<a id="fc2-itemmods-hdr9"></a>

### 要求

降低裝備需求等級。設備上顯示的需求等級是考慮了需求減少之後的。

<a id="fc2-itemmods-hdr10"></a>

### 暴擊

根據暴力破解和暴力破解的數值進行計算，以確定暴力破解率（%）。靈技增加攻擊率的效率隨著技能的增加而降低，在200靈技左右，狂暴攻擊率達到10%時幾乎停止增加。每攻擊50次，攻擊率增加約2-3%，效率也隨之下降。

在物理攻擊的情況下，斗篷武器基礎增幅150%，風暴斗篷武器基礎增幅200%，魔法增幅150%。如果增加攻擊率50，則增加攻擊率3%，如果是150%，實際傷害會增加1~1.5%，如果是200%，實際傷害會增加2~3%。如果你的蠻力極低或你的魔法職業很難找到要插入的符文，則可能會增加它。

<a id="fc2-itemmods-hdr11"></a>

### Attack Rating

增加物理技能傷害和命中率。至於對物理技能的影響程度，攻擊等級100相當於能力30到35。

<a id="fc2-itemmods-hdr12"></a>

### 施法速度增加

提高施法速度。根據施法速度增幅值計算法術施法急速（%），並決定施法速度。作為粗略的指導，施法速度增加 20 會使施法急速提高 2.5%。

<a id="fc2-itemmods-hdr13"></a>

### 資源

健康、法力、精神。增加每種資源的價值。既然是加成，那麼原值越高，效果就越低。

<a id="fc2-itemmods-hdr14"></a>

### 恢復速度

健康、法力、精神。每5秒恢復一次。不會因困難而受到處罰。

<a id="fc2-itemmods-hdr15"></a>

### 命中時恢復

Health、Mana、Spirit。使用武器普通攻擊或發動物理技能時恢復；多段技能視為一次。此效果會受到難度懲罰：惡夢（Nightmare）減少 33%，地獄（Hell）減少 50%。

<a id="fc2-itemmods-hdr16"></a>

### 擊殺時恢復

健康、法力、精神。擊敗敵人後恢復。即使你沒有擊敗它也會恢復。不會因困難而受到處罰。

<a id="fc2-itemmods-hdr17"></a>

### 受擊時恢復

Health、Mana、Spirit。受到攻擊時恢復；即使沒有被命中也會恢復。此效果會受到難度懲罰：惡夢（Nightmare）減少 33%，地獄（Hell）減少 50%。

<a id="fc2-itemmods-hdr18"></a>

### 最大資源

健康、法力、精神。將每種資源增加一個百分比。

<a id="fc2-itemmods-hdr19"></a>

### Damage 減免

物理、魔法。減去受到的傷害。由於是減法，所以原值越高，效果越低。

<a id="fc2-itemmods-hdr20"></a>

### 物理抗性

減少一定百分比受到的物理傷害。它通常附著在盾牌、頭部、軀幹、腰帶和大腿裝甲上。

<a id="fc2-itemmods-hdr21"></a>

### 属性抗性

整體、出血、毒素、奧術、閃電、火、冰。每個屬性所受到的傷害都會減少一定百分比。物理阻力不包含在總阻力中。

上限為 75%，且每個難度都會套用懲罰。合計來看，惡夢（Nightmare）需要堆到 105%，地獄（Hell）需要堆到 145% 才能達到上限。

<a id="fc2-itemmods-hdr22"></a>

### 吸收 属性

流血、毒素、奧術、閃電、火、冰。根據每種屬性所受傷害的比例恢復生命值。

<a id="fc2-itemmods-hdr23"></a>

### 狀態異常抗性

能力全開，頭暈、恐懼、麻痺、眼前一片漆黑。減少異常情況下所花費的時間。

上限為 50%，且每個難度都會套用懲罰。合計來看，惡夢（Nightmare）需要堆到 60%，地獄（Hell）需要堆到 75% 才能達到上限。

<a id="fc2-itemmods-hdr24"></a>

### Damage近戰

流血、毒素、奧術、閃電、火、冰。增加普通攻擊和物理技能傷害值。既然是加成，那麼原值越高，效果就越低。

<a id="fc2-itemmods-hdr25"></a>

### Spell Power

全部，出血，毒素，奧術，雷電，火，冰。為法術傷害加一個值。既然是加成，那麼原值越高，效果就越低。

<a id="fc2-itemmods-hdr26"></a>

### 属性Damage

所有法術，流血，毒素，奧術，閃電，火，冰。增加各屬性的傷害。雖然說的是所有法術，但體術也是有效果的。

很可能是和天賦、裝備的技能傷害強化加在同一個地方的，如果技能傷害強化累積得夠多，效率就會降低。作為粗略的指導，你可以將其視為實際傷害增加約 1/6。

<a id="fc2-itemmods-hdr27"></a>

### 降低 敵 抗性

全部，出血，毒素，奧術，雷電，火，冰。上限為75%。降低敵人的元素抗性。不影響黨員。作為粗略的指導，可以認為實際傷害增幅約為 1/3。

<a id="fc2-itemmods-hdr28"></a>

### 移動速度

提高城堡內的移動速度。上限為 500%。

<a id="fc2-itemmods-hdr29"></a>

### 獲得率

稀有掉落率、金幣獲取率、成功率。提高每次獲取率。黃金獲取率也會影響寶箱。

<a id="fc2-itemmods-hdr30"></a>

### 傷害增益 vs 類型

人形、野獸、不死族、惡魔、龍族、神秘、巨人。增加對每種怪物類型的傷害。物理技能被認為與技能強化和屬性傷害強化同框，實際傷害增幅約為1/6。魔法技能會根據乘數增加傷害，並且可能會使元素抗性和掉落加倍。

<a id="fc2-itemmods-hdr31"></a>

### Skill 強化

增加技能傷害。如果有多個，如果抽取相同的技能強化，則會相加。如果累積了足夠的屬性傷害和技能強化，效率就會降低。作為粗略的指導，你可以將其視為實際傷害增加約 1/6。

### 效果

進行普通攻擊時，根據幾率會產生附加效果。就算不打也能打。

效果傷害由固定傷害+Lv*n決定，不受抗性等影響。但基於點的效果受抗性影響，也可能造成濫用。

<a id="fc2-itemmods-hdr33"></a>

### 乙太

武器傷害提高 33%，護甲防禦提高 50%。戰鬥結束時，耐久度會以較低的速度消耗，當耐久度耗盡時，裝備就會消失。

<a id="fc2-itemmods-hdr34"></a>

### 不滅

有了不朽，乙太的耐久度就不會再降低。可以與 Fyrm 符文賦予一起給出。某些套裝和傳奇裝備保證了不朽。沒有乙太的永生沒有特殊的意義。

<a id="fc2-itemmods-hdr35"></a>

### Socket

它配備有武器、副披風、頭部和身體裝備，並且可以裝備與Socket一樣多的符文。一旦佩戴，該符文就無法去除或擦除。

## 神話製作（Craft）

- FC2 file：[mythical.html](https://atelier3.web.fc2.com/ngo/mythical.html)
- 原站 Last-Modified：`Thu, 26 Jun 2025 00:26:03 GMT`

可在鐵匠鋪的 Craft 分頁製作 Mythical 物品，也就是 Rune Word（符文組）系統。

<a id="fc2-mythical-基本"></a>

### 基本

<a id="fc2-mythical-製作方法"></a>

#### 製作方法

背包中同時持有要作為素體的裝備與所需符文，選取素體後按下製作按鈕即可製作。

製作時不需要先把符文插進素體的 Socket。

<a id="fc2-mythical-素材"></a>

### 素材

![white.jpg](/fc2-assets/ngo/img/i/white.jpg)

<a id="fc2-mythical-灰字"></a>

#### 灰字

以 Socketed 物品作為素體。這類物品的名稱通常是灰字；帶 Socket 的普通白裝則是白字。

Socket 數量必須和要求符文數量相同；Socket 過多或不足都不能使用。例如要求符文是 Gra 與 Iza 時，只能使用 2 Socket 物品，1 Socket 或 3 Socket 都不行。

<a id="fc2-mythical-mod"></a>

#### Mod

部分 Socketed 物品會帶有天賦或技能強化，這些詞綴會在製作後保留下來。

部分 Socketed 物品名稱前會標示 Superior（右圖）。武器會額外帶有 Damage 強化，防具會額外帶有物理防禦強化；這些數值會在製作完成後加算。

<a id="fc2-mythical-商店"></a>

#### 商店

除了從地城拾取，Socketed 物品也可能在藥劑師、鐵匠鋪、商人販售。

商品會依角色而不同；更新時機推測是在城鎮中或離線狀態經過 1 小時。

<a id="fc2-mythical-製作"></a>

### 製作

<a id="fc2-mythical-socket"></a>

#### Socket

製作完成後，成品的 Socket 會被消耗，無法再進行 Enchant（附魔）。

用於製作的符文效果不會以 Enchant（附魔）形式保留在成品上。

<a id="fc2-mythical-依素體繼承"></a>

#### 依素體繼承

Damage、Speed、物理防禦等基礎性能會繼承素體數值。

製作防具時，材質也依素體決定；素體是布甲，成品也會是布甲。

<a id="fc2-mythical-需求等級"></a>

#### 需求等級

Mythic 物品的需求等級會採用素體與配方兩者之中較高的數值。

因此無法用精英素材的性能做出需求等級 9 之類的裝備。

<a id="fc2-mythical-技能-mod"></a>

#### 技能 Mod

若素體帶有天賦或技能強化，成品的 Mod 抽選會固定在該職業池內進行。

素體已經帶有的天賦不會和製作新增的天賦重複，會從抽選池中排除。

素體已經帶有的技能強化則可能和製作新增的技能強化重複。

<a id="fc2-mythical-乙太"></a>

#### 乙太

若素體帶有乙太，成品會繼承乙太；但耐久度不會恢復，會維持原本數值。

<a id="fc2-rune"></a>

## 符文

- FC2 file：[rune.html](https://atelier3.web.fc2.com/ngo/rune.html)
- 原站 Last-Modified：`Mon, 23 Jun 2025 23:50:00 GMT`

可以連接到頭部和軀幹插槽上的物品，包括斗篷武器、飛行斗篷武器、弓、護身符、盾牌以及頭部和軀幹。

弓被視為斗篷武器，護身符被視為斗篷武器（只有部分是盔甲），盾牌被視為盔甲。

各難度首次通關 Lord Szarthax 時，可以從國王那裡取得獎勵；若角色達到指定等級，會保證獲得符文。未達等級時符文會被移除，請注意。

- 普通（Normal）：角色 Lv25 以上；低 Lv 符文？已確認 Lv45 符文。
- 惡夢（Nightmare）：角色 Lv50 以上；已確認 Lv25 到 Lv59 符文。
- 地獄（Hell）：角色 Lv90 以上；Lv39 到 Lv61。

<a id="fc2-runeselect-武器"></a>

### 武器

<a id="fc2-runeselect-物理-單手"></a>

#### 物理（單手）

武器Damage（Cros）＞全 Passive（Targ）＞力量（Gart）

速度快、基礎值低的單手武器，通常加算型的 Weapon Damage（Cros）最強。

不過即使是單手武器，像速度 2.5 這種偏慢的武器，效率也不算好。

Weapon Damage（Cros）很稀有，也很難收集足夠數量；作為過渡選項，可以先用 All Passive（Targ），其次是力量（Gart）。Attack Rating%（Rath）也很優秀，但用在單手武器上有點可惜。

遊俠的單手武器採用 Attack Rating%（Rath）看起來不錯，但要有把資源押在遊俠身上的心理準備。

<a id="fc2-runeselect-物理-雙手"></a>

#### 物理（雙手）

Attack Rating%（Rath）＞全 Passive（Targ）＞力量（Gart）

速度慢、基礎值高的雙手武器，Weapon Damage（Cros）效率較低，常會接近 All Passive（Targ）。因此 Attack Rating%（Rath）最強；但 Rath 很稀有，而且需要的數量也多，所以過渡選項可以先用 All Passive（Targ），其次是力量（Gart）。

例外是如果重視武器自動攻擊，Weapon Damage（Cros）會很強。All Passive（Targ）的最大值較高，但 Cros 對最低值的墊高效果較好。

<a id="fc2-runeselect-遊俠弓"></a>

#### 遊俠弓

武器傷害 (Cros)、武器速度 (Rok) > 所有被動 (Targ) > 力量 (Gart)

自由流弓較特殊，其效率受武器的基本速度和弓孔數量影響。

Sinifay的金鷂是精英套裝，擁有最好的武器傷害（Cros），武器速度（Rok）與所有被動（Targ）的效率大致相同，但在快速Reito Doku等的情況下，武器速度（Rok）超過了第三個插槽的武器傷害（Cros）效率。有關更多信息，請參閱武器 DPS 計算器。

<a id="fc2-runeselect-弓-魅惑-適用於勇利以外的實體職業"></a>

#### 弓（魅惑）適用於勇利以外的實體職業

Attack Rating%（Rath）＞全 Passive（Targ）＞力量（Gart）

可以用與物理學相同的方式來思考它。

然而，由於戰士武僧無法裝備弓，所以它變成了魅惑，而魅惑的攻擊率%（Rath）被視為護甲，所以所有被動（Targ）都排在了前面。另外，技能速度（Phar）成為魅力中的護甲效果，因此戰士僧侶無法從中受益。我為你感到難過。

<a id="fc2-runeselect-魔法"></a>

#### 魔法

抗性降低（Mael）＞属性Damage≒暴擊（Tae）

與物理不同，諸如智慧（Ruck）和智慧（Nag）之類的地位系統極其脆弱。

由於Mael很難獲得，因此Tae似乎是最安全的火力選擇。

我認為提高稀有掉落率（Marr）也是一個不錯的選擇，因為魔法咒語不能像物理咒語那樣增加火力。

另外，由於環境中的聖殿騎士很少，所以準備具有執行速度的武器（Skar）也很方便。

<a id="fc2-runeselect-命中率"></a>

#### 命中率

Attack Rating%（Rath）＞全 Passive（Targ）＞Attack Rating（Shir）＞力量（Gart）

沒有增加攻擊等級能力的魔法職業（尤其是薩滿）的命中率比較困難，所以用符文提高命中率也是提高觸發攻擊率的一個選擇。基本上，攻擊等級%（Rath）很優秀，但根據手冊只有0.5%左右的差異，所以考慮到稀有性，全被動（Targ）更安全。另請注意，攻擊等級％（Rath）是魅力中的護甲效果。過程：如果目標是提高射速，使用武器速度（Rok）調整速度也是一種選擇。

<a id="fc2-runeselect-mana-回復"></a>

#### Mana 回復

命中時 Mana 回復（Gra）

對於消耗大量法力的職業，例如戰士僧侶和十字軍，擁有擊中時恢復法力（Gra）的輔助斗篷將使即使隊伍中沒有吟遊詩人時也更容易移動。

<a id="fc2-runeselect-防具"></a>

### 防具

<a id="fc2-runeselect-物理"></a>

#### 物理

全能力值（Rath）、Rare Drop Rate（Marr）

火力方面，所有增加力量的能力（拉斯）都很優秀，但是裝甲上沒有消耗拉斯的空間，我會說這太奢侈了。如果你的抵抗力極為匱乏，屬性抗性符文當然有用。

<a id="fc2-runeselect-魔法-2"></a>

#### 魔法

Rare Drop Rate（Marr）

由於沒有放入任何東西，因此通過消除法變成稀有掉落率（Marr）。也就是說，馬爾非常罕見，所以你需要對放入其中的設備進行選擇性。我們推薦諸如莫斯比的古代王冠之類的物品，因為它們可以與各種角色一起使用。

和物理類似，如果抵抗力極度缺乏，就加入屬性抵抗符文。
