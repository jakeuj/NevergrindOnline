## 遊戲流程

- FC2 file：[chart.html](https://atelier3.web.fc2.com/ngo/chart.html)
- 原站 Last-Modified：`Tue, 13 May 2025 16:18:13 GMT`

為新玩家提供遊戲策略圖表的頁面。

如果你願意，可以新增圖像。

頁內連結：Normal / Nightmare / Hell / Heroic

<a id="fc2-chart-hdr1"></a>

### Normal

這是遊戲的第一個難度級別，你將學習基礎知識。

有第一幕到第四幕，擊敗每個區域的最後一個頭目即可解鎖下一幕。

根據已通關的區域的不同，開放的區域也會有所不同。

進展路線由你決定，但擁有不死族特殊攻擊的Crusader和Cleric通常會經過薩魯布林避風港 > 裂罅石窟 > 土勒地穴。

<a id="fc2-chart-lv1"></a>

#### Lv1～

在學院裡集結技能，獲取廣泛的Talent，不失為一個好主意。

如果你使用的職業具有嚴重的法力，請注意擊中時恢復法力或具有法力再生的裝備。

如果你夠幸運獲得格拉Rune，你可以在鐵匠處為有插槽的武器附魔。

最好專注於一個地下城的進展，而不是填充一個廣泛的範圍。從經驗值效率上來說，最好對付比你高2級左右的敵人，而如果清掉區域，敵人的等級往往會更低。

擊敗一個區域的boss也會同時解鎖其他區域的boss，所以不用擔心。

另外，盡量積極參與聚會。隨著敵人數量的增加，訓練速度會發生很大的變化。即使你聽不懂英語，進來時也只需說“hi”，離開時說“tyfp”即可。當你準備好進入地牢時，請右鍵單擊螢幕底部的臉部圖像，然後按切換：準備顯示綠色劍。

<a id="fc2-chart-lv25"></a>

#### Lv25～

此時，最好重置你在學院的Talent並將點數分配給一棵樹。

在樹上花費21點後，你將能夠獲得高級Talent，這將極大地增強你的技能。

之後，你是否想發展你的卓越才能，或再次繼承另一棵樹的衣缽，這取決於你了。

<a id="fc2-chart-lv45"></a>

#### Lv45～

此時，你可能需要考慮在學院再次重置你的Talent。

一旦你在樹上花了 41 點，你就可以在底部獲得Mastery。尤其是Mastery，寫著“效果：技能名稱”，非常出色，普通攻擊命中時，寫出的技能會自動發動。

在第四幕中，征服任意區域後，需要擊敗疾風要塞的Boss Maelentia。

擊敗他後，你將滿足在灰流峰（火山）解鎖Boss領主薩薩克斯的要求，並且你將能夠挑戰他。

打敗這兩條龍就是NGO的故事情節。

順便說一句，即使你在普通模式下擊敗了第四幕 Boss，你也無法完全解鎖灰流峰的 Boss 薩薩克斯領主。灰流峰大約有一半是開放的，其餘部分有待開發。

<a id="fc2-chart-hdr2"></a>

### Nightmare

它基本上是Normal的翻版，重複同樣的事情。與普通的區別如下。

- 玩家的元素抗性和狀態抗性降低。
- 此外，被擊中時的恢復和受到擊中時的恢復將被修正和減少。
- 敵人的物理防禦和元素抗性增加
- 給予特殊敵人（例如冠軍怪物）的模組數量已從 1 個增加到 2 個。

難度會增加，但達到上級和Mastery的玩家都很強，應該問題不大。

<a id="fc2-chart-注意屬性抗性"></a>

#### 注意屬性抗性

惡夢的元素抗性受到 30% 的懲罰。這在Nightmare中並不重要，但在下一個難度級別「Hell」中變得至關重要。所以，惡夢期可以說是一個準備元素抗性的時期。

特殊裝備可以在惡夢中收集。

卓越是40級以後的裝備上才會寫的，但比一般裝備強一級。

這裡有一些元素抵抗裝備，如果你幸運的話，你會很高興得到它們。

其中包括阿納斯塔利亞鈍器之弧、毒蛇王長袍布甲胸甲、崇敬之鍊鏈甲胸甲和砲塔星堡項圈。

還有許多優秀的盾牌和套裝裝備。然後，收集屬性抗性，如果有缺少的屬性，用稀有的頭、鞋子、項圈等進行調整，或在有插槽的盔甲上添加屬性抗性Rune。

在策略開始時，嘗試使用第四幕的Matron Maelentia專注於冰冷和閃電，並使用Szarthax勳爵專注於火焰和流血。

<a id="fc2-chart-節省黃金"></a>

#### 節省黃金

如果你把錢花在諸如在學院購買技能、擴大你的商人庫存或擴大你的銀行之類的事情上，你賺到的黃金都不會留在你的錢裡。如果你意識到不購買在學院裡不使用的技能或將它們放在次要位置，情況會好一些。

<a id="fc2-chart-hdr3"></a>

### Hell

由於難度增加而產生的增強點與夢魘相同，但應用了更困難的增強點。

元素抗性特別嚴重，從夢魘到Hell，元素抗性額外降低了40%，總共降低了70%。

例如屬性抗性為0或負數，就得做好3000到5000點傷害的準備，而當多個附魔怪出現時，同時釋放閃電類技能的情況就更多，瞬間死亡的情況並不少見。

此外，敵方Shadow Knight所使用的傷害之觸非常危險，除非它具有抗流血能力。

<a id="fc2-chart-讓我們用精神來對抗元素吧"></a>

#### 讓我們用精神來對抗元素吧

我想去Hell收集過渡Hell的裝備。

在Hell難度下，從60級以上就可以收集到被稱為菁英裝備的裝備，威力極為強大。不過，從惡夢晉升到Hell的門檻異常高，這往往是一個問題。這些都是解決問題的方法。

- 帶上巴德一起
  Bard有一個增加總抵抗力的增益，同時也大大增加了生命值，對隊伍的穩定做出了很大貢獻。此外，法師還有增加所有類型抗性的增益，聖殿騎士有增加火抗性的增益，Shaman有寒冷抗性，附魔師有增加閃電抗性的增益，Cleric有增加流血抗性的增益，Warlock有增加毒素抗性的增益。
  當敵人等級超過80級時，菁英裝備就會開始出現，所以要繞過該區域。
- 寄生、接收設備
  你可能有一些想法，但這是最簡單的。獲得精英裝備是最好的​​解決辦法。
- 惡夢中的軌道主母瑪倫蒂亞 / 薩薩克斯勳爵
  這就像在收集卓越裝備的同時試圖從寶箱中獲得精英裝備，但由於你無法從敵人那裡獲得精英裝備，所以你往往需要耐心。
- 瞄準賭博領域的精英地位
  我不推薦，因為它需要金幣，但是很多精英物品都具有出色的抵抗力，而且賭博配件只要你能買到精英，基本上是無敵的，所以很容易擴展。我想要40k的購買成本。
  有關更多信息，請參閱賭博建議。

如果你能用這些將所有屬性抗性降低到60-75%，那就沒問題了。

<a id="fc2-chart-注意稀有掉落率"></a>

#### 注意稀有掉落率

我想在屬性抗性旁邊收集它。套裝會隨著裝備的增加而自然增加，但我希望稀有掉落率即使很低也能在100%左右。誤差在200%左右開始出現，但只要該數值很小，UniqueDrop Rate就有相當大的差異。

<a id="fc2-chart-hdr4"></a>

### Heroic

如果你在Hell難度下擊敗薩薩克斯領主，Hell任務選擇畫面上將新增英雄複選框。

基本結構與Hell相同，但敵人更強大，冠軍怪物的比例也增加。

這是非常危險的，但它使收集物品和賺取經驗值變得更容易。

<a id="fc2-chart-lv95"></a>

#### Lv95～

Lv95之後升級所需的經驗值非常可笑，所以大多數玩家都是在這裡加入的。

至此，Talent似乎可以從兩種樹木中獲得Mastery。

進入英雄的標準是擁有精英套裝或類似的裝備。

就DPS而言，如果你能做到相当于75k，你就被认为是一个成熟的玩家，如果你能做到相当于100k，你就被认为是一个carry。

<a id="fc2-chart-旋轉地牢"></a>

#### 旋轉地牢

你可以在英雄模式中繞行的地下城目前幾乎已修復。

- 第二幕 - 裂罅洞窟 - 裂罅洞窟之王
  最受歡迎。首先，你可以充分利用擁有不死族特殊攻擊的Crusader和Cleric。另外，地圖相當大，出遊也方便，而且由於是區域boss，所以boss掉落也很多。
  第三幕的Thule Crypt也是同樣的不死地牢，但只是稍遜一籌，因為有很多反奧術角色。
- 第三幕 - 西隆庇護所 - 加齊恩領主
  隨著強大傳奇人物的神秘特殊攻擊的增加，它已成為一種選擇。由於奧術抗性極高，他對Crusader、Cleric等奧術職業較弱。
- 第四幕 - 疾風要塞 - 上半場任務
  因為是冰抗高火抗低的副本，所以很容易使用法師、Monk、盜賊等火屬性職業。而且，由於人形數量較多，如果延長人形特攻的話，很容易就繞過去。
  地下城的後半部並不受歡迎，因為出現了具有物理抵抗力的巨人。
- 第四幕 - 灰流峰 - 上半場任務
  這是一個火抗高、冰抗低的副本，很容易被以Druid為主的隊伍選出。
  這也是以人形為中心的，所以人形特殊攻擊是有效的。
  副本後半段，物理抗性的巨人和奧術抗性的Hell犬都是垃圾。

<a id="fc2-loot-装備収集"></a>

#### 装備収集

擁有+2佔有Talent（也附加魔法）的稀有物品非常有用，如果同時附加力量（最大+50），可以在物理職業中長期使用。此外，+3Talent樹對於某些職業很有用，例如Crusader的審判者和Templar的遠見者和元素師，它們可以用作鏈接，直到你選擇獨特。

<a id="fc2-loot-金銭効率"></a>

#### 金銭効率

從原價來看，板甲最貴，布甲最便宜。另外，就身體部位而言，胸甲是最高的，我認為裝備越重，它往往越高。另外，根據鑑定後的mod，價格會加成，而且Talent和屬性抗性都特別有針對性，所以把這些贈送的零件帶回家是划算的。

優先順序似乎是這樣的。

區分鈍器的物理和魔法的唯一方法是記住圖標，雖然你不習慣，但使用捲軸來評估它們是否有Talent可能是個好主意。物理鈍器的形狀就像它們生來就是為了用棍棒打死人的。

●雙手披風魔法鈍器（法杖）

順便說一句，當DPS超過75時，稀有的實體雙手披風的售價會穩定上漲，所以如果你能記住它，這是一個好主意。

<a id="fc2-unimon-冠軍怪物"></a>

#### 冠軍怪物

它的名字是藍色背景的怪物，是從任務中出現的怪物中被選為飛行員的。

添加模組是完全自動的。加強地位/抵抗力。

保證掉落 1 件或更好的魔法物品。

<a id="fc2-unimon-獨特的怪物"></a>

#### 獨特的怪物

它的名字是紫色背景的怪物，它出現的任務是固定的。

出現在通往地下城的途中以及通往該區域途中的 Boss 房間中的指定角色都適用。

第一個mod是固定的，第二個和第三個是mod。加強地位/抵抗力。

保證掉落 2 件或更好的魔法物品。

<a id="fc2-unimon-首領怪物"></a>

#### 首領怪物

這個名字是一個紅色背景的怪物，它出現在每個區域的最終任務中。

第一個mod是固定的，第二個和第三個是mod。加強地位/抵抗力。

他們有著高大的外表，並且可能擁有他們職業原本不具備的技能。
