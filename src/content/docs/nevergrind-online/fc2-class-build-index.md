### Talent 範例

<a id="fc2-bard-分頁-支援"></a>

#### 分頁：支援

強調對黨員的支持，增強大師（高級）正義狂想曲或迪納摩戰歌的類型。 《高級戰歌》獲得亞軍是因為它有點奇怪。

以前的正義狂想曲只對自己有效，但現在是全隊共享，所以你要確保你能把它傳給上級。抗性降低和物理傷害加成都是共享的。

<a id="fc2-bard-分鐘頁-打孔"></a>

#### 分鐘頁：打孔

這是一種使用資源 Damage 並強調 Composer 樹的打孔風格。

作曲家和大師：很難獲得電池升壓，因此從支援類型開始可能是安全的。不過，雖然是拳擊類型，但並不代表支撐力會降低那麼多。

以前的正義狂想曲只對自己有效，但現在是全隊共享，所以你要確保你能把它傳給上級。抗性降低和物理傷害加成都是共享的。另外，當你收到附魔師的相位之刃時，它也會增強你自身的火力。

只需購買一次 Superior Subverted Symphony 即可擴大目標範圍並提高可用性。

不過積分非常嚴格，如果優先考慮電池的話，97級就可以拿到，所以優先考慮哪一個就看個人喜好了。目標在第 8 級時增加。

### Talent 範例

<a id="fc2-cleric-分頁-火力-支援"></a>

#### 分頁：火力＋支援

<a id="fc2-cleric-arbiter-共通"></a>

#### Arbiter（共通）

主技能譴責之樹。

高級譴責在 25 歲時為你提供 +5 額外命中。實現起來並不困難。

也許可以選擇從布朗身上移除5，但他的傷害並沒有那麼糟糕，所以我懷疑是否值得移除它並選擇其他東西。

Mastery：提高審判者等級會減少譴責的冷卻時間，但由於原來的時間太長，即使從15級提升到25級，效果也很小，快速33揮就會變成30揮，所以我覺得不值得入手。

<a id="fc2-cleric-vestal"></a>

#### Vestal

這棵樹是與治療和增益相關的Talent的集合，可以有助於你的隊伍的穩定。

獲得譴責詩篇會增加治療傷害，以盟友為目標時會治療，以敵人為目標時會造成傷害。這是一個方便的攻擊技能，可以填補譴責冷卻時間的空白，但問題是它可能會導致操作錯誤，例如想恢復就攻擊，或想攻擊就恢復。

<a id="fc2-cleric-分頁-火力特化"></a>

#### 分頁：火力特化

<a id="fc2-cleric-arbiter-共通-2"></a>

#### Arbiter（共通）

主技能譴責之樹。

高級譴責在 25 歲時為你提供 +5 額外命中。實現起來並不困難。

也許可以選擇從布朗身上移除5，但他的傷害並沒有那麼糟糕，所以我懷疑是否值得移除它並選擇其他東西。

Mastery：提高審判者等級會減少譴責的冷卻時間，但由於原來的時間太長，即使從15級提升到25級，效果也很小，快速33揮就會變成30揮，所以我覺得不值得入手。

<a id="fc2-cleric-scion"></a>

#### Scion

相較於火力+支援型（亞維斯塔），DPS提升約10~20k。這樣單人轉比較容易。

問題是你無法獲得狂熱決心的整體效果，而且在沒有Templar的情況下移動有點困難。

各技能的增幅時間分別為：拯救15、神聖庇護所16、神聖啟示16。

Mastery：占卜效果：拯救的冷卻時間在11級時約為15.6秒，在20級時約為9.5秒，較長。

### 操作與打法

當你進入地牢時，施展[Slice:10.png]救贖聖印和[Slice:11.png]熱忱決心。

#### Elementalist

該樹以火與冰為中心，冰凍寶珠為中流砥柱。

冰凍寶珠具有當範圍內敵人較多時命中次數增加，當範圍內敵人較少時命中次數減少的特性。因此，你在戰鬥初期會受到傷害，但如果拖到最後階段，你的DPS就會大幅下降。

我覺得我作為替補選擇的技能，熔岩箭或火牆沒有太大差別。

熔岩箭是一種簡單的基於螺栓的技能，可以讓單個對手變得更好一些。火牆是一種在10秒內對所有排列的敵人造成傷害的技能，初始狀態下每秒造成一次傷害。

### Talent 範例

<a id="fc2-wizard-分頁-閃電型"></a>

#### 分頁：閃電型

閃電系火力高，但對Templar的依賴度極高。因此，很難創造一個產生火力的環境。

<a id="fc2-wizard-chaos-mage"></a>

#### Chaos Mage

它是一棵以閃電為中心的樹，擁有閃電箭，具有高 DPS。

超級閃電箭的額外命中增幅為 19/29 級，並且在達到 29 級之前火力較低。

Mastery：Oh Bard Live 增強了完整性並影響效果：閃電的冷卻時間。

|ＯＤランク|20|21|22|23|24|
|---|---|---|---|---|---|
|クールタイム|2.4|2.2|1.9|1.7|1.5|

如果你正確收集裝備並擲M，你的等級將是23。雖然在這個狀態下已經足夠強，但如果高級閃電的提升足夠並且有裝備空間，那麼它會+1，最快1.5秒。

冷卻時間1.5後，會受到此效果整體冷卻時間1.5的影響，不會再減少。

在該狀態下，裝備速度等於（或慢於）冷卻時間的武器，並在每次揮舞時發射閃電箭。

<a id="fc2-wizard-pyromancer"></a>

#### Pyromancer

它是一棵以火焰為中心的樹，具有出色的範圍傷害。

另一個選擇是將子樹設為 Ice Mage 而不是 Pyromancer，並使 Deep Freeze 更厚。火力會降低，但穩定性會因凍結而增加。當獨奏或你的隊伍較弱時有效。

<a id="fc2-wizard-分頁-火焰型"></a>

#### 分頁：火焰型

火焰系的主要優點是範圍傷害，所以前期敵人多的時候DPS很高，但是後期敵人少的時候DPS就下降了。

導致其DPS不如閃電繫穩定，火力也有些遜色。然而，與閃電類型不同的是，它不太依賴Templar，而是更依賴盜賊和Ranger，它們具有攻擊能力增益，可提高命中率。

<a id="fc2-wizard-pyromancer-2"></a>

#### Pyromancer

它是一棵以火焰為中心的樹，具有出色的範圍傷害。

火焰類型可以透過Mastery：毀滅而獲得，其效果旨在以短時間間隔重複擊中流星。

|破壊ランク|24|25|26|27|28|29|
|---|---|---|---|---|---|---|
|クールタイム|2.8|2.5|2.2|1.9|1.6|1.5|

毀滅等級的目標是28或29，這需要裝備+3-4提升。

在此狀態下，裝備速度等於（或慢於）冷卻時間的武器，並在每次揮舞時發射流星。

冷卻時間1.5後，會受到此效果整體冷卻時間1.5的影響，不會再減少。

利用分配給流星的額外點數來增強火箭和火球術的輔助火力。

高級火箭/火球額外命中增加為 19/29。

<a id="fc2-wizard-chaos-mage-2"></a>

#### Chaos Mage

它是一棵以閃光為中心的樹，但火焰類型的目的是收集鏡像和掌握：Oh Bard Live。

該圖像顯示了閃電，但你可以使用其他任何東西。

如果你是閃電系，就收集混沌法師/閃電箭，如果你是火焰系，就收集烈焰法師/流星。

伊澤人套裝在雙手披風武器中太弱了，所以需要釋放披風全Talent+3的全套效果。

除此之外，你還必須達到高級閃電和Mastery：毀滅的等級，並主要透過隱密典範和向京/戒指來收集+4。而且，伊澤人5套的Lair Dro 65強度很高，所以不太可能進一步破壞。

如果右手斗篷武器是閃電類型，則使用65紫色風暴切割者，速度1.5到1.7，閃電Talent+2。Mastery：如果Oh Bard現場等級為23，則需要仔細選擇速度為1.7的一個。

如果你是火焰系，使用被遺忘者速度1.7-1.8的68紫色串，火焰Talent+2。如果你負擔得起且不需要Talent提升，64紫色Hell刺在速度1.5（1.4是×）和亡靈特攻方面表現出色。

請注意，該裝備適合單獨使用或沒有速度增益，因此如果 PT 中有Bard（速度 +20-30%）或Ranger（+20%），你可以通過將左右手切換到神秘典範速度 2.3 至 2.6 或將其懸在空中來響應。

阿爾德隆4套可以獲得50個稀有多羅，如果技能強化匹配的話，將會是一套很均衡的裝備。

如果腰帶是68布甲紫色暗影絲網，可以獲得技能強化，但是稀有抽籤會消失。 64 特雷斯塔的玉帶，有紫色的布質外殼，也可以依Talent使用。

對於大腿腳背和鞋子，優先考慮阻力。大腿裝甲和辛電一樣堅固，但卡恩有70條布甲紫色閃亮金屬護腿。鞋子方面，有50個皮甲紫色爭霸戰之光和65個布甲紫色財團拖鞋，可以讓你以抵抗為代價獲得閱讀。

### 操作與打法

進入地下城時，施放[Scale:10.png]魔力外殼和[Scale:9.png]鏡像。

[Slice:9.png] 鏡像的屬性增強僅在護盾存在時有效，因此每次冷卻結束時都需要使用它。

戰鬥時，只需在敵人聚集的地方使用[圖片:7.png]流星和[圖片:4.png]火球，並重複擊打[圖片:0.png]火焰箭直至冷卻結束。對於閃電類型，只需點擊 [Scale:7.png] Meteor，然後重複點擊 [Scale:2.png] Lightning Bold。

[圖：11.png] Deep Freeze 可以有效對抗危險的敵人，例如狂暴和嗜血。

如果你的健康變得危險，請使用冰凍屏障來恢復。
