### Damage強化

這是一個僅適用於武器的MOD，並且加強了武器的傷害。裝備上顯示的數字和DPS是應用傷害增強後的數字和DPS。

<a id="fc2-itemmods-hdr2"></a>

### 攻擊速度增加

這是一個只能應用在武器上的MOD，可以提高武器的速度。裝備上顯示的數字和DPS是加上攻擊速度提升後的。

<a id="fc2-itemmods-hdr3"></a>

### 阻塞率

這是一個僅適用於盾牌方塊的模組，並指示方塊的擴展率。這個數字只是命中率，格擋時的基礎傷害削減率為25%。坦克職業也增加20%的格擋率。

<a id="fc2-itemmods-hdr4"></a>

### 區塊率增加

這是一個僅適用於盾牌並增強格擋率的模組。設備上顯示的阻塞率是應用阻塞率增加後的值。

<a id="fc2-itemmods-hdr5"></a>

### 物理理防禦強化

這是只適用於護甲、強化物理防禦的MOD。裝備上顯示的物理防禦是應用物理防禦強化後的。

<a id="fc2-itemmods-hdr6"></a>

### Talent

增加角色的適用Talent。如果有多個Talent，則不會抽取相同的Talent。它通常附著在武器、盾牌、潛水艇、頭、胸甲、斗篷、腰帶、項圈和戒律上。 Talent是Talent的機器翻譯，唯一的差異是是否為複數。

- 擁有Talent
  增加所有類別的所有Talent。
- 完整性 類別名稱 Talent
  增加適用職業的所有Talent。
- Talent樹
  晉升適用Talent樹中的所有Talent。
- 屬性Talent
  流血、毒素、奧術、閃電、火和冰。增加具有適合所有職業屬性的Talent。對於具有多種屬性的技能，只會應用一種。例如，Druid的龍捲風是冰冷和奧術技能，但僅適用於冰冷Talent。
- Talent
  增加適用Talent。

<a id="fc2-itemmods-hdr7"></a>

全能力，力量，耐力，敏捷，敏捷，智慧，智慧，魅力。

<a id="fc2-itemmods-hdr8"></a>

### Passive

所有被动技能，攻击、防御、剑披斩、剑披钝器、刺击、射箭、空手道披风、剑披斩、剑披钝器、挥舞、双重攻击、闪避、招架、反击、变异、召唤、召唤。

武器被動非常重要，它極大地影響物理技能傷害和命中率。作為一個粗略的指導，如果武器是被動15，物理技能傷害將增加3.3%。

法術被動不會增加那麼多法術傷害，所以有點奇怪。改造傾向於影響恢復等支援系統，召喚傾向於影響範圍系統，召喚傾向於影響單一系統。

<a id="fc2-itemmods-hdr9"></a>

### 要求

降低裝備需求等級。設備上顯示的需求等級是考慮了需求減少後的需求等級。

<a id="fc2-itemmods-hdr10"></a>

### 暴擊

根據敏捷度和暴擊值進行計算，確定暴擊率（%）。因敏捷而增加暴擊率的效率隨著敏捷度的增加而降低，而當敏捷在200左右時，在10%暴擊率時幾乎停止增加。每50次暴擊，暴擊率增加約2-3%，效率也隨之下降。

就物理而言，單手隱形武器暴擊傷害基本提升150%，雙手隱形武器提升200%，魔法提升150%。如果暴擊增加50，暴擊率增加3%，如果是150%，實際傷害會增加1-1.5%，如果是200%，實際傷害會增加2-3%。如果你的爆擊率極低，或者你的魔法職業在選擇Rune時遇到困難，你可能需要增加它。

<a id="fc2-itemmods-hdr11"></a>

### 攻撃能力

增加物理技能傷害和命中率。至於對物理技能的影響程度，攻擊能力為100，力量相當於30到35。

<a id="fc2-itemmods-hdr12"></a>

### 施法速度增加

提高施法速度。根據施法速度增幅值計算法術施法急速（%），並決定施法速度。作為粗略的指導，施法速度增加 20 會使施法急速提高 2.5%。

<a id="fc2-itemmods-hdr13"></a>

### 資源

健康、法力、精神。增加每種資源的價值。由於是相加，所以效果會隨著原值的增加而減弱。

<a id="fc2-itemmods-hdr14"></a>

### 再生

健康、法力、精神。每5秒恢復一次。不會因困難而受到處罰。

<a id="fc2-itemmods-hdr15"></a>

### 命中時恢復

健康、法力、精神。使用武器的普通攻擊或使用物理技能時恢復。多階段技能視為使用一次。依難度等級會有懲罰，惡夢減少33%，Hell減少50%。

<a id="fc2-itemmods-hdr16"></a>

### 擊殺時恢復

健康、法力、精神。擊敗敵人後恢復。即使你沒有擊敗它也會恢復。不會因困難而受到處罰。

<a id="fc2-itemmods-hdr17"></a>

### 受擊時恢復

健康、法力、精神。受到攻擊時恢復。即使沒有被擊中也會恢復。依難度等級會有懲罰，惡夢減少33%，Hell減少50%。

<a id="fc2-itemmods-hdr18"></a>

### 最大資源

健康、法力、精神。將每種資源增加一個百分比。

<a id="fc2-itemmods-hdr19"></a>

### 傷害減少

物理、魔法。減去受到的傷害。由於是減法，因此效果會隨著原始值的增加而減少。

<a id="fc2-itemmods-hdr20"></a>

### 物理理抗性

物理傷害減少一定百分比。它通常附在盾牌、頭部、胸甲、腰帶和大腿甲上。

<a id="fc2-itemmods-hdr21"></a>

### 属性抗性

整體、流血、有毒、奧術、閃電、火焰、寒冷。每個屬性所受到的傷害都會減少一定百分比。物理阻力不包含在總阻力中。

最高為75%，且每個難度等級都有懲罰。總共來說，惡夢最高為 105%，Hell最高為 145%。

<a id="fc2-itemmods-hdr22"></a>

### 吸收屬性

流血、毒素、奧術、閃電、火和冰。根據每種屬性所受傷害的比例恢復生命值。

<a id="fc2-itemmods-hdr23"></a>

### 狀態異常抗性

全力、眩暈、恐懼、麻痺、沉默。減少異常情況下所花費的時間。

最高為50%，且每個難度等級都有懲罰。總共來說，惡夢最多為 60%，Hell最多為 75%。

<a id="fc2-itemmods-hdr24"></a>

### Damage近接

流血、毒素、奧術、閃電、火和冰。為普通攻擊和物理技能傷害增加數值。由於是相加，所以效果會隨著原值的增加而減弱。

<a id="fc2-itemmods-hdr25"></a>

### 法術強度

所有，流血，劇毒，奧術，閃電，火焰，寒冷。增加法術傷害值。由於是相加，所以效果會隨著原值的增加而減弱。

<a id="fc2-itemmods-hdr26"></a>

### 属性Damage

所有法術，流血，毒素，奧術，閃電，火，冰。增加各屬性的傷害。雖然說的是所有法術，但對身體技能也有影響。

很可能是和Talent、裝備的技能傷害強化加在同一個地方的，如果技能傷害強化累積得夠多，效率就會降低。作為粗略的指導，你可以將其視為實際傷害增加約 1/6。

<a id="fc2-itemmods-hdr27"></a>

### 降低 敵 抗性

所有，流血，劇毒，奧術，閃電，火焰，寒冷。上限為75%。降低敵人的元素抗性。不影響黨員。作為粗略的指導，可以認為實際傷害增幅約為 1/3。

<a id="fc2-itemmods-hdr28"></a>

### 移動速度

提高地牢中的移動速度。上限為 500%。

<a id="fc2-itemmods-hdr29"></a>

### 獲得率

稀有掉落率、金幣獲得率、經驗值獲得率。提高每次獲取率。黃金獲取率也會影響寶箱。

<a id="fc2-itemmods-hdr30"></a>

### 傷害增強與類型

類人生物、野獸、亡靈、惡魔、龍皮、神秘主義者、巨人。增加對每種怪物類型的傷害。物理技能看起來和技能強化、屬性傷害強化屬於同一類，實際傷害增幅大概1/6左右。魔法技能會根據乘數增加傷害，並且可能會使元素抗性和掉落加倍。

<a id="fc2-itemmods-hdr31"></a>

### Skill強化

增加技能傷害。如果有多個，如果抽取相同的技能強化，則會相加。如果累積了足夠的屬性傷害和技能強化，效率就會降低。作為粗略的指導，你可以將其視為實際傷害增加約 1/6。

### 効果

根據普通攻擊的機率，會產生額外的效果。就算不打也能打。

效果傷害由固定傷害+Lv*n決定，不受抗性等影響。但點系效果受抗性影響，也會造成暴擊。

<a id="fc2-itemmods-hdr33"></a>

### 乙醚

武器傷害提高 33%，護甲防禦提高 50%。戰鬥結束時，耐久度會以較低的速度消耗，當耐久度耗盡時，裝備就會消失。

### Socket

## Craft

- FC2 file：[mythical.html](https://atelier3.web.fc2.com/ngo/mythical.html)
- 原站 Last-Modified：`Thu, 26 Jun 2025 00:26:03 GMT`

你可以在鐵匠工藝標籤中製作神話物品。也稱為Rune字。

<a id="fc2-mythical-基本"></a>

### 基本

<a id="fc2-mythical-作成方法"></a>

#### 作成方法

透過將成為基礎身體的物品和必要的Rune放入庫存中，選擇基礎身體並按下創建按鈕來創建。

建立 Socket 時無需將 Rune 插入其中。

<a id="fc2-mythical-素材"></a>

### 素材

<a id="fc2-mythical-灰字"></a>

#### 灰字

使用帶有Socket的物品（名稱為灰色/白色帶Socket）作為基礎。

需要與請求Rune相同數量的Socket，超過或少於Socket數量都不能使用。例如，如果請求Rune為GRA和IZA，則只能使用2個插槽的實體插槽，而不能使用1和3。

<a id="fc2-mythical-mod"></a>

#### Mod

一些帶有插槽的物品附有Talent和技能增強功能，這些增強功能在製作時會保留下來。

有些插槽物品的名字前面寫有「高級」（右圖），如果是武器，則會獲得額外的傷害增強，如果是盔甲，則會獲得額外的物理防禦增強。這些將在生產後添加。

<a id="fc2-mythical-商店"></a>

#### 商店

除了在地下城中拾取外，帶孔物品還可以在藥劑師、鐵匠和商人處出售。

每個角色的產品都會改變，更新時間是在城裡1小時後還是線下？

<a id="fc2-mythical-製作"></a>

### 製作

<a id="fc2-mythical-socket"></a>

#### Socket

製作的物品的插槽會被消耗且無法附魔。

用於製作的Rune的效果並未附魔。

<a id="fc2-mythical-素体準拠"></a>

#### 素体準拠

傷害、速度、物理防禦等基礎性能繼承基體數值。

製作鎧甲時，材質是根據本體來的，如果本體是布甲，那麼成品也將是布甲。

<a id="fc2-mythical-需求等級"></a>

#### 需求等級

對於神話需求等級，優先考慮基體或配方的較高值。

因此，不可能生產出具有精英材料性能的需求等級9的裝備。

<a id="fc2-mythical-skillmod"></a>

#### SkillMod

如果身體獲得Talent或技能增強，則所生產物品的模組抽籤將固定為該職業。

已經附著在身體上的Talent不會與透過生產添加的Talent重疊，並且會被排除在抽籤之外。

已經賦予身體的技能增強與透過生產添加的技能增強重疊。

<a id="fc2-mythical-乙醚"></a>

#### 乙醚

如果以太附著在身體上，它會被攜帶，但不會恢復耐久度，保持不變。

<a id="fc2-runeselect-武器"></a>

### 武器

<a id="fc2-runeselect-物理理-片手套"></a>

#### 物理理（片手套）

速度高、基礎值低的單手斗篷往往擁有最強的武器傷害（十字架），是一種加成武器。

不過，即使是單手使用，像極速2.5這樣的慢速效率也不是很高。

武器傷害（Cross）很少見，很難收集，所以選項都是被動（Targ），其次是力量（Gart）作為連結。攻擊能力%（拉斯）也很優秀，但單手使用就可惜了。

Ranger的斗篷使用攻擊能力%（拉斯）似乎不錯，但你需要做好和Ranger一起自殺的準備。

<a id="fc2-runeselect-物理理-両手套"></a>

#### 物理理（両手套）

速度低、基本值高的雙手斗篷，武器傷害（交叉）效率較低，往往與所有被動（塔格）處於同一水平。因此，攻擊能力%（拉斯）最強。不過攻擊能力%（拉斯）很少見，需要大量，所以連線的選項都是被動（塔格），其次是力量（加特）。

例外的是，如果你重視武器的自動攻擊，那麼武器傷害（十字）就很強。 All Passive (Targ) 的最大值較高，但 Cros 的最小值較高。

<a id="fc2-runeselect-ranger弓"></a>

#### Ranger弓

遊騎兵弓比較特殊，其效率取決於武器的基本速度和弓孔數量。

Sinifay的金鷂是精英套裝，擁有最好的武器傷害（Cros），武器速度（Rok）與所有被動（Targ）的效率大致相同，但在快速Reito Doku等的情況下，武器速度（Rok）超過了第三個插槽的武器傷害（Cros）效率。有關更多信息，請參閱武器 DPS 計算器。

<a id="fc2-runeselect-弓-魅惑-適用於ranger以外的實體職業"></a>

#### 弓（魅惑）適用於Ranger以外的實體職業

與物理（雙手）相同的想法就可以了。

不過，Monk無法裝備弓，所以就變成了魅惑，而魅惑的攻擊力%(Rath)被視為護甲，所以所有被動(Targ)都在上面。另外，技能速度（Phar）在魅惑中成為護甲效果，因此Monk無法從中受益。我為你感到難過。

<a id="fc2-runeselect-魔法"></a>

#### 魔法

與物理不同，諸如智慧（Ruck）和智慧（Nag）之類的狀態系統極其脆弱。

反退化（Mael）很難獲得，所以如果你的目標是火力，臨界（Tae）似乎是最安全的選擇。

魔法Rune預計不會像物理那樣增加火力，所以我認為增加稀有掉落率（Marr）也是一個不錯的選擇。

另外，由於環境中聖殿騎士很少，因此準備具有執行速度的武器（Skar）很方便。

<a id="fc2-runeselect-命中率"></a>

#### 命中率

沒有增加攻擊能力Talent的魔法職業（尤其是法師）的命中率比較困難，所以用Rune提高命中率也是提高觸發攻擊率的一個選擇。基本上，攻擊能力%（拉斯）很優秀，但基於單手斗篷的差異只有0.5%左右，所以考慮到稀有性，全被動（塔格）比較安全。另外，請注意攻擊能力%（Rath）是魅力中的護甲效果。過程：如果目標是提高射速，使用武器速度（Rok）調整速度也是一種選擇。

<a id="fc2-runeselect-mana回復"></a>

#### Mana回復

對於消耗大量魔力的職業，例如Monk和Crusader，擁有擊中時恢復魔力（格拉）的副裝備，即使隊伍中沒有Bard，也能更輕鬆地移動。

<a id="fc2-runeselect-防具"></a>

### 防具

<a id="fc2-runeselect-物理理"></a>

#### 物理理

就火力而言，增加力量（拉斯）的能力很出色，但是裝甲上沒有消耗拉斯的空間，所以我會說它太奢侈了。如果你的抵抗力極度匱乏，元素抵抗Rune當然有用。

<a id="fc2-runeselect-魔法-2"></a>

#### 魔法

由於沒有放入任何東西，因此通過消除法變成稀有掉落率（Marr）。然而，馬爾相當罕見，所以你需要對放入其中的設備進行選擇性。推薦像莫斯比的古代王冠這樣的物品，因為它們可以與各種角色一起使用。

和物理類似，如果抵抗力極度不足，就附加屬性抵抗Rune。
