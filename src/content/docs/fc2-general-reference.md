## 遊戲流程

- FC2 file：[chart.html](https://atelier3.web.fc2.com/ngo/chart.html)
- 原站 Last-Modified：`Tue, 13 May 2025 16:18:13 GMT`

為新玩家提供遊戲策略圖表的頁面。

如果你願意，可以新增圖像。

頁內連結：普通（Normal） / 惡夢（Nightmare） / 地獄（Hell） / 英雄（Heroic）

<a id="fc2-chart-hdr1"></a>

### 普通（Normal）

普通（Normal）是遊戲的第一個難度，你會在這裡學習基礎流程。

有第一幕到第四幕，擊敗每個區域的最後一個頭目即可解鎖下一幕。

根據清除區域的不同，打開的區域也會有所不同。

推進路線可以自由選擇；如果是擁有不死族特攻的十字軍與牧師，通常會走 Salubrin Haven > Riven Grotto > Thule Crypt。

<a id="fc2-chart-lv1"></a>

#### Lv1～

在學校獲得廣泛的技能和廣泛的天賦是一個好主意。

如果你使用的职业具有严重的法力，请注意击中时恢复法力或具有法力恢复速度的装备。

如果幸運拿到 Gra 符文，可以到鐵匠鋪把它附到有插槽的武器上，推進會輕鬆很多。

最好將城堡的進程縮小到一小部分，而不是填滿大範圍的區域。对付比你高2级左右的敌人会更有效率，而且如果你清理了该区域，敌人的等级往往会更低。

擊敗一個區域的boss也會同時解鎖其他區域的boss，所以不用擔心。

另外，盡量積極參與聚會。隨著敵人數量的增加，訓練速度會發生很大的變化。即使你聽不懂英語，進來時也只需說“hi”，離開時說“tyfp”即可。當你準備好進入城堡時，請右鍵單擊螢幕底部的臉部圖像，然後按切換：準備顯示綠色劍。

<a id="fc2-chart-lv25"></a>

#### Lv25～

到了這個階段，建議先到學院重置天賦，集中把點數投入同一條天賦樹。

在同一條天賦樹投入 21 點後，就能取得 Superior 天賦，技能會獲得大幅強化。

之後要繼續提高 Superior 天賦，或改投資另一條天賦樹，就看你的偏好。

<a id="fc2-chart-lv45"></a>

#### Lv45～

到了這個階段，也可以考慮再次到學院重置天賦。

在同一條天賦樹投入 41 點後，就能取得最下方的 Mastery。特別是寫著「效果：技能名稱」的 Mastery 很優秀，普通攻擊命中時會自動觸發標示的技能。

Act IV 中，任一區域通關後，還需要擊敗 Galeblast Fortress（冰山）的 Boss Matron Maelentia。

擊敗她後，才會滿足 Ashenflow Peak（火山）Boss Lord Szarthax 的解鎖條件，並可以開始挑戰。

打敗這兩條龍就是NGO的故事情節。

順帶一提，即使只在普通（Normal）擊敗 Act IV Boss，也不會完整解鎖 Ashenflow Peak 的 Boss Lord Szarthax。大約只會開放 Ashenflow Peak 的一半，剩下區域仍需要繼續推進。

<a id="fc2-chart-hdr2"></a>

### 惡夢（Nightmare）

惡夢（Nightmare）基本上是普通（Normal）的重跑，需要重複同樣的流程；和普通（Normal）的差異如下。

- 玩家的元素抗性和狀態抗性降低。
- 此外，被擊中時的恢復和受到擊中時的恢復將被修正和減少。
- 敵人的物理防禦和元素抗性增加
- 給予特殊敵人（例如冠軍怪物）的模組數量已從 1 個增加到 2 個。

難度會提高，但取得 Superior 與 Mastery 的玩家通常已經很強，理論上不會太卡。

<a id="fc2-chart-注意屬性抗性"></a>

#### 注意屬性抗性

惡夢（Nightmare）會讓元素抗性受到 30% 懲罰。這在惡夢（Nightmare）階段還不算致命，但到了下一個難度地獄（Hell）會變成最重要的門檻。因此，惡夢（Nightmare）可以視為提前準備元素抗性的階段。

惡夢（Nightmare）可以開始收集 Exceptional 裝備。

卓越是40級以後的裝備上才會寫的，但比一般裝備強一級。

這裡有一些元素抵抗裝備，如果你幸運的話，你會很高興得到它們。

其中包括阿納斯塔利亞鈍器之弧、毒蛇王長袍布甲胸甲、崇敬之鍊鏈甲胸甲和砲塔星堡項圈。

盾牌與套裝裝備也有許多優秀選擇。接著要開始堆元素抗性；如果某個屬性缺口很大，可以用稀有頭盔、靴子、項鍊等部位補足，或在有插槽的防具上鑲嵌屬性抗性符文。

在策略開始時，嘗試使用第四幕的Matron Maelentia專注於冰冷和閃電，並使用Szarthax勳爵專注於火焰和流血。

<a id="fc2-chart-節省黃金"></a>

#### 節省黃金

如果你把錢花在學校購買技能、擴充商人背包或擴張銀行等事情上，你賺到的黃金都不會留在你的錢裡。如果你記住，如果你不購買在學校用不到的技能，你可以把它們放在次要位置，這樣會好一點。

<a id="fc2-chart-hdr3"></a>

### 地獄（Hell）

地獄（Hell）的強化方向和惡夢（Nightmare）相同，但數值與壓力都會更嚴苛。

元素抗性尤其嚴苛；從惡夢（Nightmare）進入地獄（Hell）後，元素抗性會再降低 40%，合計等於降低 70%。

例如屬性抗性為0或負數，就得做好3000到5000點傷害的準備，而當多個附魔怪出現時，同時釋放閃電類技能的情況就更多，瞬間死亡的情況並不少見。

另外，敌方暗影骑士使用的伤害之触除非具有抗流血能力，否则极其危险。

<a id="fc2-chart-讓我們用精神來對抗元素吧"></a>

#### 讓我們用精神來對抗元素吧

常見卡點是「想去地獄（Hell）收集能轉進地獄（Hell）的裝備」。

地獄（Hell）難度可以收集 60 級以後的 Elite 裝備，威力非常強。不過，從惡夢（Nightmare）升到地獄（Hell）的門檻異常高，這裡很容易成為卡關點。常見解法如下。

- 帶上吟遊詩人一起
  吟游诗人有一个增益，可以增加总抵抗力，也可以大大增加生命值，对队伍的稳定有很大贡献。另外，薩滿還有增加全抗的buff，神社騎士有增加火抗的buff，薩摩男有寒冷抗性，隨從法師有閃電抗性，牧師有增加流血抗性的buff，魔法師有增加毒素抗性的buff。
  请注意，当敌人的等级超过 80 级时，精英装备就会开始出现，因此请围绕该区域运行。
- 寄生、接收設備
  你可能有一些想法，但這是最簡單的。獲得精英裝備是最好的​​解決辦法。
- 周回惡夢（Nightmare）的 Matron Maelentia / Lord Szarthax
  一邊收集 Exceptional 裝備，一邊從寶箱瞄準 Elite 裝備；因為敵人本身幾乎不會正常掉 Elite 裝備，所以通常需要耐心。
- 瞄準賭博領域的精英地位
  需要金幣，所以不推薦，但是大部分精英物品都具有極好的抗性，而賭博配件只要能買精英基本上是無敵的，所以很容易擴展。我想要40k的購買成本。
  有關更多信息，請參閱賭博建議。

如果你能用這些將所有屬性抗性降低到60-75%，那就沒問題了。

<a id="fc2-chart-注意稀有掉落率"></a>

#### 注意稀有掉落率

我想在屬性抗性旁邊收集它。套裝會隨著裝備的增加而自然增加，但我希望稀有掉落率即使很低也能在100%左右。誤差從200%左右開始出現，但只要該數值很小，Drop Rate就有相當大的差異。

<a id="fc2-chart-hdr4"></a>

### 英雄（Heroic）

擊敗地獄（Hell）難度的 Lord Szarthax 後，地獄（Hell）的任務選擇畫面會新增英雄（Heroic）勾選框。

基本結構與地獄（Hell）相同，但敵人會更強，Champion monster 的比例也會增加。

雖然非常危險，但相對地更適合收集物品與刷經驗。

<a id="fc2-chart-lv95"></a>

#### Lv95～

Lv95 以後升級所需經驗值非常誇張，多數玩家大約會在這裡開始匯合周回。

到了這個階段，天賦通常會變成取得兩條天賦樹 Mastery 的配置。

進入英雄（Heroic）的門檻，大致是湊齊 Elite Set 或同等級裝備。

以 DPS 來說，能打出約 75k 的貢獻就算成熟玩家；約 100k 則可視為 carry 端。

<a id="fc2-chart-周回地城"></a>

#### 周回地城

英雄（Heroic）常見的周回地城目前幾乎已固定。

- Act.II - Riven Grotto - King of Riven Grotto
  最受歡迎。理由是能充分發揮十字軍與牧師的不死族特攻；地圖大小適中、路線好跑，又是區域 Boss，因此 Boss 掉落也多。
  Act.III 的 Thule Crypt 同樣是不死族地城，但 Arcane 抗性的怪物很多，整體只算下位替代。
- Act.III - Sylong Sanctuary - Lord Gazzion
  因強力傳奇裝備增加 Mystical 特攻，逐漸成為可選地城。怪物 Arcane 抗性非常高，十字軍、牧師等 Arcane 職業比較不擅長。
- Act.IV - Galeblast Fortress - 前半任務
  這裡冰冷抗性高、火焰抗性低，巫師、武僧、盜賊等火焰屬性職業比較好刷。Humanoid 數量多，堆 Humanoid 特攻也能提高周回效率。
  地城後半會出現具物理抗性的 Giant，因此不受歡迎。
- Act.IV - Ashenflow Peak - 前半任務
  這裡火焰抗性高、冰冷抗性低，常被以德魯伊為核心的隊伍選用。
  這裡也以 Humanoid 為主，因此 Humanoid 特攻有效。
  地城後半有物理抗性的 Giant，以及 Arcane 抗性的 Cerberus，很難處理。

## 常見問題

- FC2 file：[faq.html](https://atelier3.web.fc2.com/ngo/faq.html)

<a id="fc2-faq-1010"></a>

### 關於角色建立

種族與能力值分配可能會讓人猶豫，但差異不大，不需要太在意。最重要的是頭像，會直接影響遊玩動力，請慎重選擇。

<a id="fc2-faq-1099"></a>

### 天梯與永久角色的差異

天梯會設置排行榜，用角色等級競爭排名。賽季結束時，天梯角色會移到永久角色池。過去曾有不掉傳奇、不能使用博物館等差異，但已經隨更新調整。

<a id="fc2-faq-1140"></a>

### 關於賽季

可繼承的要素只有銀行擴充格。賽季以三個月為單位，賽季結束後天梯角色會移到永久角色池，身上的裝備、Gold 與背包內容也會一起移過去，但不能再轉回天梯；銀行內物品會消失。

<a id="fc2-faq-1174"></a>

### 不知道如何進入地城

從城下町左側的酒類商店選擇任務並進入。

<a id="fc2-faq-1333"></a>

### 鑑定很麻煩

城鎮中央偏左的藥劑店可以一鍵鑑定。持有未鑑定物品時，藥劑店視窗左側會出現「全部鑑定」按鈕。

<a id="fc2-faq-1500"></a>

### 買賣物品很麻煩

使用 Ctrl + 左鍵即可快速買賣。

<a id="fc2-faq-2416"></a>

### 如何擴充背包

你可以透過在城下町右側的商人處支付金幣來完成此操作。

<a id="fc2-faq-2519"></a>

### 如何鑲嵌符文

到城鎮右側的鐵匠鋪即可鑲嵌。符文只能裝在有插槽的裝備上，而且一旦鑲嵌就無法拆下。

<a id="fc2-faq-2704"></a>

### 如何重置天賦

你可以透過在城下町右上角的學校支付金幣來做到這一點。

<a id="fc2-faq-3395"></a>

### 如何分享物品

把游標移到物品上，按 Shift + 左鍵即可貼到聊天中。

<a id="fc2-faq-4124"></a>

### 如何比較裝備

把游標移到物品上，按 Shift 即可和目前裝備中的物品比較。

<a id="fc2-faq-4445"></a>

### 不知道如何顯示綠色劍圖示

加入隊伍後右鍵自己的頭像，選擇「Toggle: Ready」即可顯示綠色劍圖示，用來告知隊長你已準備完成。

<a id="fc2-faq-4590"></a>

### 想更換技能快捷鍵

右鍵即可替換。

<a id="fc2-faq-5998"></a>

### 想知道聊天指令

在聊天輸入 `/help` 可以叫出指令列表。

<a id="fc2-faq-5999"></a>

### 加入公會有什麼好處

沒有實質加成。勉強要說的話，就是可以使用公會聊天。

<a id="fc2-faq-6008"></a>

### 地城獎勵類型是什麼

Boss 掉落最上方的物品，會是畫面顯示的兩種物品類型之一。武器依種類區分，鈍器還分物理與魔法；防具依部位區分，不區分材質。

<a id="fc2-faq-6073"></a>

### 裝備上的天賦與技能文字顏色代表什麼

這是為了讓你容易判斷效果適用於哪個職業。例如巫師是紅色、牧師是黃色。

<a id="fc2-faq-6347"></a>

### 效果是什麼

效果是武器自動攻擊時觸發的追加效果。武器最下方寫的效果會依機率觸發；天賦底部的 Mastery 等來源取得的效果，基本上依冷卻時間觸發。

<a id="fc2-faq-6555"></a>

### 乙太與不滅是什麼

乙太裝備的圖示會呈半透明，乙太數值（100）代表耐久度。武器會提高基礎攻擊力 33%，防具會提高基礎防禦力 50%，但有低機率讓耐久減少約 1 到 10；耐久歸零時裝備會消失。帶有不滅的裝備不會減少耐久。

<a id="fc2-faq-7019"></a>

### Rapid Attack 是什麼

Rapid Attack 會讓武器自動攻擊加速。有些職業可以透過天賦底部的 Mastery 取得。

<a id="fc2-faq-8421"></a>

### 想大致判斷 Boss 房位置

用滑鼠滾輪縮小地圖，再拖曳捲動看看。Boss 房常常在地圖空白較大的方向深處。

<a id="fc2-faq-8989"></a>

### 想讓地城移動更輕鬆

如果你是按住左鍵移動，可以按住左鍵把游標拖到頭像附近再放開，角色會維持前進狀態。遊戲也有自動移動鍵，可參考快捷鍵設定。

<a id="fc2-faq-9338"></a>

### 遊戲逐漸變卡

使用「Camp and Logout」回到標題畫面通常就會恢復。

<a id="fc2-faq-9634"></a>

### 有時會無法操作

可能是由於伺服器端的某些情況，在日本時間上午 8:00 和上午 8:30 出現了一個神秘的凍結時間。

<a id="fc2-faq-9793"></a>

### 物品消失了

這款遊戲有物品或空白格重疊在同一格的問題。物品資訊與位置資訊通常仍在，可以先把另一個物品放到該格再移開，接著重新登入，有時物品就會重新出現。

### 英文/縮寫

aa / afk, brb, bio / cya / dot / dps / gz, gratz / hc / hero / ic / idk / imma / lfg / lfm / mf / nm / np / pl / proc / pst / r, rdy / sc / sk / ty, thx, tyvm, tyfp / wb, cb / wtb, wtt, wts / yw

<a id="fc2-english-0461"></a>

#### aa

auto Attack的縮寫，指使用武器進行自動攻擊。

<a id="fc2-english-1197"></a>

#### afk, brb, bio

遠離鍵盤的縮寫，意思是遠離鍵盤。此外，經常使用“be right back”。 3m afk 和 3min brb，有 3 分鐘座位。 「bio」是「biological」的縮寫，用於上廁所時。

<a id="fc2-english-1549"></a>

#### cya

這是“see you Again”的縮寫，因為“c”和“see”的發音相同。

<a id="fc2-english-1845"></a>

#### dot

這是「damage over time」的縮寫，指的是隨著時間的推移逐漸造成傷害的一種形式。當我想到毒素和出血時，我腦海中就會浮現出一個畫面。

<a id="fc2-english-2117"></a>

#### dps

Damage Per Second 的縮寫，指每秒造成的傷害量。也是由此衍生而來，指的是火力職業。

<a id="fc2-english-2381"></a>

#### gz, gratz

祝賀的縮寫，意思是恭喜。形式逐漸縮寫為congrats（congratz）→grats（gratz）→gz。當你升職或成為騙子時，可以安全地說「GZ」。

<a id="fc2-english-2751"></a>

#### hc

「硬核」的縮寫，指的是硬核模式，當你死亡時你的角色會被刪除。或是下面的英雄模式。

<a id="fc2-english-2972"></a>

#### hero

「英雄」的縮寫，指的是地獄難度中擊敗火龍解鎖的英雄模式，用於招募隊員時使用。有時縮寫為 hc。

<a id="fc2-english-3232"></a>

#### ic

I see 是縮寫，表示我明白。

<a id="fc2-english-3331"></a>

#### idk

「我不知道」的縮寫，意思是「我不知道」。

<a id="fc2-english-3472"></a>

#### imma

「I'mgoingto」的縮寫，意思是「我要去。」源自 I'mgoon。

<a id="fc2-english-3577"></a>

#### lfg

「尋找團體」的縮寫，意思是你想加入一個聚會。供想要入黨的一方使用。

<a id="fc2-english-3881"></a>

#### lfm

「尋找更多」的縮寫，意思是尋找黨員。 lf2m 吟遊詩人坦克正在尋找兩名 PT 成員，並且想要一名吟遊詩人和一名坦克。

<a id="fc2-english-4071"></a>

#### mf

magic find的縮寫，日文寫為Rare Drop Rate。

<a id="fc2-english-4269"></a>

#### nm

「Nightmare」的縮寫，指的是惡夢難度。招募PT時使用。

<a id="fc2-english-4493"></a>

#### np

「no Problem」的縮寫意思是沒問題或沒有問題。用作對感謝或道歉的回應。

<a id="fc2-english-4679"></a>

#### pl

透過省略代練，你可以一次培養出強大的角色。又稱水產養殖。

<a id="fc2-english-5081"></a>

#### proc

是指在特定動作後以一定比例給出的附加效果。用NGO術語來說，是指賦予特定的獨特武器的「效果」部分。

<a id="fc2-english-5361"></a>

#### pst

「please sendtell」的縮寫，意思是我正在等待訊息或個人聊天。

<a id="fc2-english-5583"></a>

#### r, rdy

它是“ready”的縮寫，在準備好時使用。詢問時，使用 r?回答時使用 r。現在，右鍵單擊臉部圖像並出現劍圖標，它會通知你準備工作已完成。

<a id="fc2-english-5699"></a>

#### sc

Soft core 的縮寫，意思是正常。那些不是硬核的人。

<a id="fc2-english-5841"></a>

#### sk

指暗影騎士。

## 值得帶回的物品

- FC2 file：[loot.html](https://atelier3.web.fc2.com/ngo/loot.html)
- 原站 Last-Modified：`Sun, 01 Jun 2025 08:54:55 GMT`

討論哪些物品值得撿回城鎮，以及賣價上的金錢效率。

Craft 相關系統實裝不久，本頁暫不討論。

<a id="fc2-loot-稀有-rare-等級以下"></a>

### 稀有（Rare）等級以下

<a id="fc2-loot-裝備收集"></a>

#### 裝備收集

![neck.jpg](/fc2-assets/ngo/img/i/neck.jpg)

帶有 +2 所有天賦的稀有（Rare）項鍊非常有用（Magic 也有機會出現）。如果同時帶力量（最高 +50），物理職業可以用很久。另外，+3 天賦樹對特定職業很有價值，例如十字軍的 Judicator、聖殿騎士的 Visionary / Elementalist，在撿到對應獨特裝備前都能當過渡裝。

![hand.jpg](/fc2-assets/ngo/img/i/hand.jpg)

稀有（Rare）手套也可能出現 +2 天賦樹。術士只有傳奇（Legendary）能出 +2，聖殿騎士則只有稀有（Rare）能出 +2。板甲職業也可以把它當成抽到 Marshal Gauntlets Iniquity 目標天賦樹之前的過渡品。不過天賦詞綴非常少見，而且也有還可以的替代品，所以重要度沒有那麼高。

<a id="fc2-loot-金錢效率"></a>

#### 金錢效率

以基礎價格來看，板甲最高、布甲最低。以部位來說胴體最高；外觀看起來越重的裝備，價格通常也越高。鑑定後，某些 Mod 會讓價格增加，其中天賦與屬性抗性特別值得注意。把可能出現這些詞綴的部位帶回去，金錢效率會比較好。

稀有（Rare）魔法鈍器（單手 / 雙手） > 胴體 > 盾牌、頭部、手部、腿甲、靴子

大致上優先順序應該是這樣。

物理鈍器和魔法鈍器只能靠圖示分辨；不熟時也可以用鑑定卷軸確認有沒有天賦。看起來像天生就是拿來敲人的那種，就是物理鈍器。

● 單手魔法鈍器（Focus）

![focus.png](/fc2-assets/ngo/img/i/focus.png)

● 雙手魔法鈍器（Stave）

![stave.png](/fc2-assets/ngo/img/i/stave.png)

順帶一提，稀有（Rare）物理雙手鈍器在 DPS 超過 75 左右後，賣價會穩定偏高；如果記得住，建議記起來。

<a id="fc2-loot-獨特-unique-等級以上"></a>

### 獨特（Unique）等級以上

<a id="fc2-loot-裝備收集-金錢效率"></a>

#### 裝備收集 / 金錢效率

只要是優秀（Exceptional）以上，無論裝備留用或賣錢效率，基本上都可以全部帶回。

後期普通（Normal）裝備基本上可以丟掉沒關係。不過魔法雙手鈍器即使是普通（Normal），所有裝備都會帶天賦，所以能賣不錯的價格。其他普通（Normal）獨特裝備則看你是否記得哪些會出天賦或屬性抗性；能辨認的話就帶回。

![bow.jpg](/fc2-assets/ngo/img/i/bow.jpg)

此外，普通（Normal）獨特 / 套裝當中也有少數該帶回的例外，主要是弓。

Stormcaller Bow 和 Demetrium's Ballista 的最大 Socket 數可能留到最終裝備，因此請務必帶回。

如果不確定帶回的獨特或套裝該留還是該賣，可以參考各職代表技能、各職嚴選獨特裝備，以及各職 Build 介紹頁。

## 賭博建議

- FC2 file：[gambling.html](https://atelier3.web.fc2.com/ngo/gambling.html)
- 原站 Last-Modified：`Wed, 09 Apr 2025 16:47:42 GMT`

說明城鎮右側商人提供的 Gambling（賭博）功能。

基本上，銀行擴充告一段落後，Gold 通常會開始溢出。

這時就很適合開始 Gambling。歡迎來到泥沼。

<a id="fc2-gambling-規則"></a>

### 規則

- 進入地城或重新登入會刷新商品列表。
- 武器與防具的圖示是固定的；請購買與目標圖示相同的品項。
- Charm、Necklace、Ring 的圖示不是固定的（隨機）。
- Charm、Necklace、Ring 沒有棕框或橙框，但把游標移上去即可判斷 Normal / Exceptional / Elite。
- 傳奇（Legendary）也會出現，但通常需要數百次嘗試。

<a id="fc2-gambling-推薦商品"></a>

### 推薦商品

以下介紹眾多品項中特別值得優先購買的目標。

<a id="fc2-gambling-飾品類-exceptional-elite"></a>

#### 飾品類（Exceptional / Elite）

![amulet.jpg](/fc2-assets/ngo/img/g/amulet.jpg)

Charm、Necklace、Ring 等飾品在 Exceptional / Elite 階級沒有 Rare 等級，因此會保證出獨特（Unique）以上，算是很划算。反過來說，其他品項多半像把 Gold 丟進水溝。基本上以購買這些飾品為主，其他品項當作順手加買。

Necklace 目標是天賦很強的 Shimmering Fragment 與 Sayanim Kaleidoscope。Ring 雖然不像 Necklace 那麼強，但抗性優秀的 Black Swan Band、Heaven's Wing 很有用。這也能用來瞄準套裝；尤其 Necklace 的抽選池很深，需要多累積嘗試次數。

<a id="fc2-gambling-charlatan-s-crest-shako-elite"></a>

#### Charlatan's Crest（Shako, Elite）

![shako.jpg](/fc2-assets/ngo/img/g/shako.jpg)

獨特裝備 Charlatan's Crest 同時具備 +2 所有天賦、屬性傷害強化與高 Rare Drop Rate 三個重點，適合許多職業採用。理想上可以為常用屬性傷害 roll 各準備一頂。

另外也有機會順便瞄準傳奇裝備 Mosby's Ancient Crown，是值得賭的品項。

<a id="fc2-gambling-cryptic-paragon-haniwa-elite"></a>

#### Cryptic Paragon（Haniwa, Elite）

![haniwa.jpg](/fc2-assets/ngo/img/g/haniwa.jpg)

這一項主要是在瞄準獨特裝備 Cryptic Paragon。Haniwa 俗稱的價值在於 +2 所有天賦、隨機天賦、全屬性抗性、全 Spell Damage 與不死生物特攻都集中在同一件裝備上，法系職業很容易用得上；真正難的是找到天賦與技能 roll 都契合的那一把。

參考：Haniwa 試行 132 次，稀有 127 個、獨特 5 個（3.78%）。

<a id="fc2-gambling-normal-exceptional-的實用品"></a>

#### Normal～Exceptional 的實用品

![noex.jpg](/fc2-assets/ngo/img/g/noex.jpg)

高 Rare Drop Rate 的 Trek of Glory，以及 6 Socket 的 Demetrium's Ballista，多少都不嫌多。

<a id="fc2-gambling-特定獨特部位-elite"></a>

#### 特定獨特部位（Elite）

![uni.jpg](/fc2-assets/ngo/img/g/uni.jpg)

Elite 皮甲胴體 Courtier's Trunk、Elite 板甲手套 Marshal Gauntlets Iniquity 的天賦很優秀，是容易放進多職業畢業裝的獨特目標。

Goliath Husk 也能同時瞄準傳奇 Hide of the Hydra。

<a id="fc2-gambling-特定套裝部位-elite"></a>

#### 特定套裝部位（Elite）

![set.jpg](/fc2-assets/ngo/img/g/set.jpg)

如果想追求更高的天賦或技能傷害，也可以購買與自身套裝裝備相同圖示的品項。

特別推薦有隨機天賦抽選的裝備，例如附魔師武器、聖殿騎士武器、遊俠武器等。

參考：Eldritch Orb 試行 125 次，稀有 119 個、獨特 5 個（4%）、套裝 1 個（0.8%）。

<a id="fc2-gambling-特定傳奇-elite"></a>

#### 特定傳奇（Elite）

![leg.jpg](/fc2-assets/ngo/img/g/leg.jpg)

這會是相當漫長且容易空手而歸的旅程，但如果有熱情，也可以把購買目標放在傳奇。

看起來會套用 Legendary Boost，因此最近變得稍微實際一些。

Firmament Staff of the Crystal Sea、Thoth Mindlink Staff 等都是強力的傳奇裝備。
