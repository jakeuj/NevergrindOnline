## 吟遊詩人

- FC2 file：[bard.html](https://atelier3.web.fc2.com/ngo/bard.html)
- 原站 Last-Modified：`Sun, 19 Apr 2026 21:44:07 GMT`

強緩衝有助於 PT 的穩定性。但老實說，這樣做讓我昏昏欲睡。

如果你把它做成拳打造型，它就會變成一個瘋狂的角色，在使用buff的同時，還能產生堪比普通攻擊的火力職業的DPS。

圖示為[圖：BRD.png]，職業代表色為淺紫色。

Set 裝備：Emissary / Volaki / Fansy

### 天賦範例

<a id="fc2-bard-分頁-支援"></a>

#### 分頁：支援

強調對黨員的支持，增強大師（高級）正義狂想曲或迪納摩戰歌的類型。 《高級戰歌》獲得亞軍是因為它有點奇怪。

以前的正義狂想曲只對自己有效，現在是全隊共享，所以你想把它交給上級。具有抗降解和抗物理損傷紅色雙重優點。

另一方面，如果我們假設上正狂想曲的物理傷害紅色增益為每級1.5%，那麼這將是物理技能約0.25%的增強。最好決定擴展哪一個以及擴展多少以適合你自己的風格。

只需購買一次 Superior Subverted Symphony 即可擴大目標範圍並提高可用性。

額外的分數將進入高級半音階奏鳴曲。

<a id="fc2-bard-分鐘頁-打孔"></a>

#### 分鐘頁：打孔

它是使用資源 Damage 並強調 Composer 樹的打孔類型。

作曲家和大師：很難獲得電池升壓，因此從支援類型開始可能是安全的。不過，雖然是拳擊類型，但並不代表支撐力會降低那麼多。

以前的正義狂想曲只對自己有效，現在是全隊共享，所以你想把它交給上級。具有抗降解和抗物理損傷紅色雙重優點。另外，當你從附件大師那裡收到相位之刃時，它也會增加你自己的火力。

只需購買一次 Superior Subverted Symphony 即可擴大目標範圍並提高可用性。

不過評分非常嚴格，如果優先考慮電池的話，會得到97的等級，所以優先考慮哪一個就看個人喜好了。目標在第 8 級時增加。

主技能譴責之樹。

<a id="fc2-cleric-vestal"></a>

#### Vestal

一棵具有治療和增益相關天賦的樹，可以為隊伍的穩定做出貢獻。

獲得譴責詩篇會增加治療傷害，以盟友為目標時會治療，以敵人為目標時會造成傷害。這是一個方便的攻擊技能，可以填補譴責冷卻時間的空白，但問題是它可能會導致操作錯誤，例如想恢復就攻擊，或想攻擊就恢復。

#### 分頁：火力特化

<a id="fc2-cleric-arbiter-共通-2"></a>

#### Arbiter（共通）

主技能譴責之樹。

<a id="fc2-cleric-scion"></a>

#### Scion

相較於火力+支援型（次要技術維斯塔），DPS提升約10~20k。這樣單人轉比較容易。

問題是你無法得到熱忱決心的概括，而且在沒有神殿騎士的情況下移動有點困難。

各技能的增幅時間分別為：拯救15、神聖庇護所16、神聖啟示16。

Mastery：占卜效果：拯救的冷卻時間在11級時約為15.6秒，在20級時約為9.5秒，較長。

如果你正確收集裝備並擲M，你的等級將是23。雖然在這個狀態下已經足夠強大，但如果超級閃電的提升足夠並且有裝備零件的空間，那麼最快會在1.5秒+1。

冷卻時間1.5後，不會進一步減少，因為它會受到此效果的全域冷卻時間1.5的影響。

在該狀態下，裝備速度等於（或慢於）冷卻時間的武器並每次發射閃電箭。

<a id="fc2-wizard-pyromancer"></a>

#### Pyromancer

它是一棵專注於火焰並具有出色區域傷害的樹。

另一個選擇是使用冰法師而不是烈焰法師作為你的輔助斗篷，並更頻繁地獲得深度凍結。火力會降低，但穩定性會因凍結而增加。當獨奏或你的隊伍較弱時有效。

<a id="fc2-wizard-分頁-火焰型"></a>

#### 分頁：火焰型

火焰系的主要優點是範圍傷害，所以前期敵人多的時候DPS很高，但是後期敵人少的時候DPS就下降了。

導致其DPS不如閃電繫穩定，火力也有些遜色。然而，與閃電類型不同的是，它不太依賴神殿騎士，而是更依賴盜賊和花花公子，他們擁有攻擊等級增益，可以提高命中率。

#### Pyromancer

它是一棵專注於火焰並具有出色區域傷害的樹。

火焰類型可以透過Mastery：毀滅來獲得，目的是在短時間內重複使用流星。

|冷靜時間|2.8|2.5|2.2|1.9|1.6|1.5|

毀滅等級的目標是28或29，這需要裝備+3-4提升。

在該狀態下，裝備速度等於（或慢於）冷卻時間的武器並每次發射流星。

冷卻時間1.5後，不會進一步減少，因為它會受到此效果的全域冷卻時間1.5的影響。

利用分配給流星的額外點數來增強火箭和火球術的輔助火力。

高級火箭/火球的紅色命中增加為 19/29。

### 操作與打法

進入地城時，施放[Scale:10.png]魔力外殼和[Scale:9.png]鏡像。

[Slice:9.png] 鏡像的屬性增強僅在護盾存在時有效，因此每次冷卻結束時都需要使用它。

戰鬥時，只需在敵人聚集的地方使用[圖片:7.png]流星和[圖片:4.png]火球，並重複擊打[圖片:0.png]火焰箭直至冷卻結束。對於閃電類型，只需點擊 [Scale:7.png] Meteor，然後重複點擊 [Scale:2.png] Lightning Bold。

[圖：11.png] Deep Freeze 可以有效對抗危險的敵人，例如狂暴和嗜血。

如果你的健康變得危險，請使用冰凍屏障來恢復。
